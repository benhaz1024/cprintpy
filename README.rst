cprintpy
======

Introduction
------------
cprintpy means color print for python.

Usage
-----
.. code: python
        from cprintpy.cprint import cprint
        cprint('R', 'hello world!')

- param 1:
 color code:
 R  red
 G  green
 B  blue
 P  purple
 C  cyan
 H  highlight

- param 2:
 your string to be colored.
